**Company:** CODTECH IT SOLUTIONS PVT. LTD.

## Project

This project is developed using Microsoft Power BI as part of my Data Analytics Internship at CODTECH IT SOLUTIONS PVT. LTD.

The dashboard is created using an e-commerce dataset to understand sales performance and customer-related insights through different visualizations.

## Tools Used

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX

## Dashboard Includes

- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Monthly Revenue Trend
- Revenue by Product Category
- Revenue by Age Group
- Payment Method Distribution
- Customer Distribution by Gender
- Customer Churn Analysis
- Slicers for filtering data
